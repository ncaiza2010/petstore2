===============================================================
     INSTALACIÓN DEL AMBIENTE DE PRUEBAS CON ECLIPSE Y KARATE
===============================================================

Este archivo README proporciona instrucciones detalladas para configurar el ambiente de pruebas utilizando Eclipse y Karate.

1. Requisitos del Sistema:
   - Sistema operativo: Windows 10, macOS 10.14 o superior, Ubuntu 20.04 LTS o distribuciones similares.
   - Java Development Kit (JDK) 8 o superior.

2. Enlaces de Descarga:
   - JDK: [Descargar JDK] https://www.oracle.com/java/technologies/javase/jdk18-archive-downloads.html 
   - Eclipse: [Descargar Eclipse](https://www.eclipse.org/downloads/)
   - Proyecto PetStore2 (Norma Caiza ejercicio 2): https://github.com/ncaiza2010/petstore2
   
3. Instrucciones de Instalación:
   a. Descargar Eclipse desde el enlace proporcionado.
   b. Instalar Eclipse siguiendo las instrucciones del instalador.
 
4. Configuración del Entorno:
    - Abrir Eclipse e instalar las siguientes librerías ( menú Help, opción Eclipse Marketplace:)
		- Cucumber Eclipse Plugin 1.0.0.202106240526
		- JunitTools 1.1.0

5 Importa el proyecto petstore2 del ejercicio 2: 
	- Ejecutar el proyecto y revisar el reporte de resultados html generado.
	- file:///XXXX/eclipse-workspace/proyecto/test-projectKarate/target/karate-reports/karate-summary.html
	

6. Referencias:
   - [Documentación de Eclipse](https://www.eclipse.org/documentation/)
   - [Documentación de JUnit 5](https://junit.org/junit5/docs/current/user-guide/)
   - [Documentación de Karate](https://github.com/intuit/karate/wiki)
